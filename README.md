# StarStacker V1.0.0
### A lightweight dropped item stacking plugin written for StarRaid.Minehut.gg which runs on 1.21.10 (paper)
You can start by downloading the latest version from the releases page. The plugin was written by 3add and can be used for your own projects when including credits.
## Instalation
- Download from Releases
- Drag and Drop into plugins folder
## Bug Reports
You can report a bug to me by either opening an issue on the tracker or sliding me a DM on discord under `not3add#0000`, please make sure you are using the latest version of paper (1.21.10 #155) as this is what the plugin was written for. I might add more features on request but I will not spend time supporting other platforms and/or lower mc version.
